# PartTracker

PartTracker is a production-ready solution supporting Volvo's operations, built with .NET.

It provides core functionality for:

- **Inplant material deliveries**
- **Fleet Management**
- **Visualisations**

---

## Installation

Clone the repository and restore dependencies:

```bash
-- Initial Pull

git clone https://github.com/volvo-cars/PartTracker.git
cd PartTracker
touch appsettings.json
touch appsettings.Development.json

-- Run

dotnet restore
dotnet build
dotnet watch run
```
